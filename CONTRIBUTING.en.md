# How to contribute to Granite

→ [Leer en español](CONTRIBUTING.md)

Welcome. If you made it here, you probably already sense that the tools 
to properly manage wilderness territories don't exist yet.

This is where we build them.

---

## Who we are

We are not a company. We don't have an onboarding department.
We are a small group working from a remote valley in Chilean Patagonia,
convinced that open source code can help protect what remains of the planet.

If that resonates with you, you are welcome here.

---

## How to participate

**If you have an idea**
Open an [Issue](../../issues). Describe what you see, what's missing, what you could build.
You don't need permission. You don't need a perfect format.

**If you want to write code**
1. Fork the repository
2. Create a branch with a descriptive name (`sensor-integration`, `visitor-dashboard`, etc.)
3. Write, test, document
4. Open a Pull Request and tell us what you
